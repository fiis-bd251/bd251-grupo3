## **Requerimientos funcionales**

### Caso de Uso 1: Registrar paquete recibido

| **Campo**            | **Descripción**                                                                 |
|----------------------|---------------------------------------------------------------------------------|
| **Objetivo**         | Permitir que el responsable de almacén registre los paquetes que han llegado al almacén para su posterior seguimiento. |
| **Descripción**      | Proceso mediante el cual el responsable de almacén registra la llegada de un paquete al almacén. |
| **Actor Primario**   | Responsable de almacén                                                           |
| **Actor Secundario** | Sistema de gestión de paquetes                                                   |
| **Precondiciones**   | El paquete debe haber llegado al almacén.                                        |
| **Flujo de Eventos** | 1. El responsable de almacén accede al sistema.<br> 2. Selecciona "Registrar Paquete Recibido".<br> 3. Ingresa el número de seguimiento del paquete.<br> 4. El sistema verifica y muestra la información asociada.<br> 5. El responsable confirma los detalles.<br> 6. El sistema guarda los datos. |
| **Postcondición**    | El paquete queda registrado en el sistema de seguimiento.                       |
